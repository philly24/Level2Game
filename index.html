1. Why I Made This Game

I made this application to entertain and cause a positive response I also made this game to express my creativity, while creating a simple but unique game for the player.

2. What I Learned

What I have learned from the from usability test is that it may take many test trials in order to find bugs especially if the program have a randomization aspect. For instances, I incorporated a random generator that randomly change the position of my food. Therefore, it took many test trials in order to find the bug regarding my randomizer. I have also learned that in order to make a good game, the times needs to be long enough to keep the user interested, but short enough so its not repetitive.
